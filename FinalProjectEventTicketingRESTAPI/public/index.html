🎫 Event Ticketing API

v1.0.0 · Live

A RESTful API for browsing events, booking tickets, and managing an event platform. Built with Node.js, Express, MongoDB, and JWT authentication.

Authentication

  • POST/api/auth/register
  • POST/api/auth/login
  • GET/api/auth/meprotected

Events

  • GET/api/events?category= & ?date=
  • GET/api/events/:id
  • POST/api/eventsadmin only
  • PUT/api/events/:idadmin only
  • DEL/api/events/:idadmin only

Bookings

  • GET/api/bookingsprotected
  • GET/api/bookings/:idprotected
  • POST/api/bookingsprotected
  • GET/api/bookings/validate/:qradmin only

Admin Dashboard

  • GET/api/admin/dashboardadmin only
